###  逆序数对计数问题

将原数组A排序得到数组B，得到B中最小数在原数组中的位置，此数可与其前面的数组成逆序数对

[7,4,8,2]   
[2,4,7,8]  
2可与前面的7,4,8组成逆序数对

```python

def InversePairs(list):
    count = 0
    copy = list.copy()
    copy.sort()
    for i in range(len(copy)):
        # 得到其在原数组中的索引，然后再将此数从原数组移除
        count += list.index(copy[i])
        list.remove(copy[i])
    return count

```
