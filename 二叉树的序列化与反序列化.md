### 题目描述

介绍二叉树的序列化和反序列化

### 思路解析

##### 前序遍历

```python

class Node:
    def __init__(self, data):
        self.data = data
        self.left = None
        self.right = None


# 前序遍历  序列化
def serialByPre(head):
    if not head:
        return '#_'

    res = str(head.data) + '_'
    res += serialByPre(head.left)
    res += serialByPre(head.right)

    return res


# 前序遍历  反序列化
def deSerialByPre(str):
    str = list(str.split('_'))

    head_data = str.pop(0)
    if head_data == '#':
        return None
    head = Node(head_data)
    head.left = deSerialByPre(str)
    head.right = deSerialByPre(str)

    return head
```
