### 题目描述

给定一个整型正方形矩阵matrix， 请把该矩阵调整成顺时针旋转90度的样子。

【要求】 额外空间复杂度为O(1)。


### 解题思路

如下矩阵

[[1,2,3]
 [4,5,6]
 [7,8,9]]
 
- 将1先保存到temp中，7放到1的位置，9放到7的位置，3放到9的位置，temp放到3的位置，本次旋转完成。
 
 - 2,6,8,4 按照上述方式进行
 
 - 外层完成后进入内层，左上角的行列加一，右下角的行列减一，即到内层位置,循环上述操作，直到最中间一个数。

```python

def rotate(matrix):
    # 左上角行列
    tR, tC = 0, 0
    # 右下角行列
    dR, dC = len(matrix) - 1, len(matrix) - 1

    while tR < dR:
        rotateEdge(matrix, tR, tC, dR, dC)
        tR += 1
        tC += 1
        dR -= 1
        dC -= 1
    return matrix

def rotateEdge(matrix, tR, tC, dR, dC):
    times = dR - tR
    for i in range(times):
        # 替换顺时针进行
        temp = matrix[tR][tC + i]
        matrix[tR][tC + i] = matrix[dR - i][tC]
        matrix[dR - i][tC] = matrix[dR][dC - i]
        matrix[dR][dC - i] = matrix[tR + i][dC]
        matrix[tR + i][dC] = temp

# for test
import numpy as np
matrix = list(np.arange(1, 10).reshape((3, 3)))
print(matrix)
print(rotate(matrix))

```
